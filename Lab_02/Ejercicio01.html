Control de Tamaño de Texto

Tamaño actual: 16px

La estructura es basicaente HTML (título, botones, texto). Con CSS le di un estilo moderno yo creo, lo esencial es JavaScript donde incluso añadí control con las teclas + y - ... Básicamente, HTML para la forma, CSS para el estilo y JavaScript para que funcione.

Este es un texto que puse de ejemplo cuyo tamaño puede ser modificado usando los botones que puse en la parte superior. El sistema que implemente permite aumentar y disminuir poco a poco el tamaño de la fuente, manteniendo un rango entre 12px y 24px para la legibilidad.
